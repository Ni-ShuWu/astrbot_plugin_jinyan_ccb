# 禁言嘲讽插件

AstrBot 插件，当群内有用户被禁言时，自动发送一条嘲讽消息。

## 使用方法

1. 将本插件放入 AstrBot 的插件目录
2. 在 AstrBot 后台启用插件
3. 当有人被管理员禁言时，机器人会自动发嘲讽

## 效果示例

- 哈哈哈哈，张三 被禁言了，经典！
- 张三 的嘴被管理员锁住了，钥匙呢？扔了！
- 叮！张三 的麦克风已被管理员关闭，发言请排队

## 配置

插件开箱即用，无需额外配置。

嘲讽语录可在 `main.py` 的 `MOCK_MESSAGES` 列表中自定义，支持 `{user}`（被禁言用户昵称）和 `{duration}`（禁言时长）两个变量。

## 注意事项

- 仅支持 OneBot 协议平台（如 NapCatQQ、LLOneBot 等）
- 解禁事件和全员禁言不会触发嘲讽
