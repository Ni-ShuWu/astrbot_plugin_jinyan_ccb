# 禁言嘲讽插件

AstrBot 插件，当群内有用户被禁言时，自动发送一条嘲讽消息。

## 功能

- 当群成员被管理员禁言时，机器人自动在群里发送一句随机嘲讽语录
- 内置 **200 条** 嘲讽文案，每次随机抽取，拒绝重复感
- 支持 AstrBot 插件配置界面**配置黑名单群号**，无需重启即可生效
- 支持通过指令快捷管理黑名单
- 仅对 `ban` 类型、时长 > 0、非全员禁言的事件触发

## 使用方法

1. 将本插件放入 AstrBot 的 `plugins` 目录
2. 在 AstrBot 后台启用「禁言嘲讽」插件
3. 配置黑名单（可选）：在 AstrBot → 插件管理 → 禁言嘲讽 → 配置 中编辑
4. 当有人被管理员禁言时，机器人会自动发嘲讽

## 配置说明

插件的配置项定义在 `_conf_schema.json`，可在 AstrBot 后台「插件配置」中直接编辑：

| 配置项 | 类型 | 说明 |
| --- | --- | --- |
| `blacklist_groups` | list[str] | 不发送嘲讽的群号列表，每行一个群号 |
| `enable_at_all` | bool | 是否在嘲讽消息中 @ 被禁言用户 |

黑名单的修改会立即写入 AstrBot 的插件配置文件并生效。

## 指令

虽然可以通过配置界面管理黑名单，插件也提供了指令版本，方便在群里操作：

```
/禁言嘲讽黑名单 add 123456     添加群 123456 到黑名单
/禁言嘲讽黑名单 remove 123456  从黑名单移除群 123456
/禁言嘲讽黑名单 list           查看当前黑名单
```

通过指令添加/移除的黑名单也会同步写入 AstrBot 插件配置。

## 效果示例

- 哈哈哈哈，张三 被禁言了，经典！
- 张三 的嘴被管理员锁住了，钥匙呢？扔了！
- 叮！张三 的麦克风已被管理员关闭，发言请排队
- ……（共 200 条随机文案）

## 自定义文案

嘲讽文案位于 `messages.py` 的 `MOCK_MESSAGES` 列表，支持两个占位符：

- `{user}`：被禁言用户的昵称
- `{duration}`：禁言时长（自动格式化为 秒/分钟/小时/天）

## 注意事项

- 仅支持 OneBot 协议平台（如 NapCatQQ、LLOneBot 等）
- 解禁事件和全员禁言不会触发嘲讽
- 兼容 AstrBot 插件配置系统，配置改动会自动保存到 `config.json`
